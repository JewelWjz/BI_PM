## PDF 白皮书说明

P1 和 P3 的 PDF 文件因体积较大（21 MB / 67 MB）未放入仓库，需从本地读取。

**启动 BI 竞品分析时，系统会在交互式引导中提示用户提供 PDF 的本地路径。**

团队成员可从以下位置获取文件（shareCRM_BI_Workspace 工作区用户）：

```
05_Resources_and_Knowledge/Evaluations_and_Insights/纷享销客BI智能分析平台产品白皮书（精华版）_202601.pdf
05_Resources_and_Knowledge/Evaluations_and_Insights/纷享销客BI智能分析平台产品白皮书（内部版）_202601.pdf
```

如没有工作区访问权限，请联系团队负责人获取文件。

> **注意**：官网手册链接地址文件中列出了各模块帮助文档 URL，当需要某功能的精确交互细节或最新更新内容时，用 WebFetch 打开对应链接从网页正文提取内容，比从 PDF 中推断更准确。

## 使用建议

- **快速分析**：从 P0 CSV 获取精确数字 + P1 白皮书精华版了解产品定位，可覆盖大多数竞品对比场景
- **深度分析**：在快速分析基础上，通过 P2 官网手册链接获取最新交互细节，P3 内部版补充技术架构细节
- **功能数量类问题**：以 P0 CSV 为最终权威，不使用文档/帮助中心中可能过时的数字
